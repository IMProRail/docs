# IMProRail Docs Repository

In deze repository worden de gepubliceerde snapshots van informatiemodellen en gerelateerde documentatie beheerd. Het doel van deze repository is om een centraal en versieerbaar overzicht te bieden van de formeel gepubliceerde artefacten binnen IMProRail.

Per domein worden hier de meest recente versies van informatiemodellen, praktijkrichtlijnen en aanvullende documentatie beschikbaar gesteld. Historische versies blijven behouden, zodat de ontwikkeling van modellen en richtlijnen in de tijd navolgbaar blijft.

## Domeinen

### Real Time Traffic Plan (RTTP)
- [CIM RTTP](https://improrail.github.io/docs/rttp/rttp)
- [Praktijkrichtlijn Scenario's](https://improrail.github.io/docs/RTTP/scenarios/)
